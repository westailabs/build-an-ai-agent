## What This Module Covers

We install OpenClaw on bare metal Ubuntu 24.04, wire it up as a systemd service, and send your first message through the agent. You'll understand the difference between running an agent in a container vs. directly on the host, when each makes sense, and why we default to bare metal for production workloads. By the end, you have a live agent that survives reboots and auto-restarts on failure.

**Topics:**
- OpenClaw installation and initial config
- systemd service unit design for agent processes
- Bare metal vs. container tradeoffs for agent runtimes
- First-run checklist and smoke test
