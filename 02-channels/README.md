## What This Module Covers

We connect your agent to the real world via Telegram and Discord. You'll create bot tokens, configure channel routing, understand how messages flow from a user's phone to the LLM and back, and set up the pairing process that binds a channel to a specific agent identity. We also cover multi-channel routing — how one agent can listen on multiple surfaces without message storms or duplication.

**Topics:**
- Telegram Bot API setup, webhooks vs. polling, bot token security
- Discord application creation, intent flags, slash commands
- Channel pairing and routing in OpenClaw
- Message deduplication and ordering guarantees (or lack thereof)
